# Рекомендации по выбору компонентов

## Сравнительная таблица компонентов

### Основной компьютер

| Компонент | Производительность | Цена | Рекомендация |
|-----------|-------------------|------|--------------|
| Raspberry Pi 4B (4GB) | Средняя | Низкая | Для простых задач |
| Raspberry Pi 5 (8GB) | Высокая | Средняя | Бюджетный вариант |
| **Beelink Mini PC** | **Очень высокая** | **Средняя** | **✅ Рекомендуется (лучшая производительность)** |
| NVIDIA Jetson Nano | Очень высокая (GPU) | Высокая | Для ML задач с GPU |
| Intel NUC | Очень высокая | Высокая | Для промышленного использования |

**Выбор:** Beelink Mini PC - оптимальный вариант для высокой производительности ROS2, SLAM и обработки видео. Значительно мощнее Raspberry Pi благодаря x86 архитектуре и большему количеству RAM.

### Контроллер STM32

| Модель | Производительность | Память | Цена | Рекомендация |
|--------|-------------------|--------|------|--------------|
| STM32F407VGT6 | 168 MHz | 1MB/192KB | Низкая | **Рекомендуется для начала** |
| STM32H743VIT6 | 480 MHz | 2MB/1MB | Средняя | Для сложных задач |

**Выбор:** STM32F407 - достаточно для управления моторами и чтения энкодеров. Может быть обновлен позже при необходимости.

### Драйверы моторов

| Драйвер | Ток | Напряжение | Цена | Применение |
|---------|-----|-----------|------|------------|
| L298N | 2A | 5-35V | Низкая | Малые моторы |
| DRV8833 | 1.5A | 2.7-10.8V | Низкая | **Рекомендуется** |
| BTS7960 | 43A | 6-27V | Средняя | Мощные моторы |

**Выбор:** DRV8833 - хороший баланс для большинства роботов весом до 5-7 кг.

### Моторы

**Рекомендации:**
- Pololu 37D Metal Gearmotor с энкодерами
- DFRobot 25GA370 с энкодерами
- Требования: минимум 100 RPM при 12V, встроенные энкодеры

### Камера 360°

**Варианты:**

1. **Insta360 ONE X2** (дорого, качественно)
   - Разрешение: 5.7K
   - Преимущества: Отличное качество, ПО для обработки
   - Недостатки: Высокая цена (~$300)

2. **Множественные камеры** (дешево, требует обработки)
   - 2-4 веб-камеры с широким углом (например, Logitech C920)
   - Преимущества: Дешево, гибкость
   - Недостатки: Требуется калибровка и стыковка

3. **Raspberry Pi Camera Module v3** (компромисс)
   - Разрешение: 12MP
   - Преимущества: Низкая цена, интеграция с Pi
   - Недостатки: Один объектив (не 360°), но широкоугольный

**Выбор:** Для начала - Raspberry Pi Camera Module v3, позже можно добавить вторую камеру или перейти на специализированную 360° камеру.

### Лидар MOP3

**Характеристики (ориентировочные):**
- Дальность: ~8-12 метров
- Угол обзора: 360°
- Частота сканирования: ~10 Гц
- Интерфейс: UART (3.3V или 5V)

**Адаптер USB-UART:**
- CP2102 или CH340 (доступные и надежные)
- Убедитесь, что работает на 3.3V или 5V в зависимости от лидара

### Маяк (Beacon)

**Варианты:**

1. **UWB (Ultra-Wideband)**
   - Точность: ~10 см
   - Дальность: ~10-50 м
   - Пример: Decawave DW1000, ESP32 с UWB модулем
   - **Рекомендация:** Для точной навигации

2. **Bluetooth iBeacon**
   - Точность: ~1-3 м
   - Дальность: ~5-30 м
   - Пример: ESP32 с Bluetooth
   - **Рекомендация:** Для начала (проще и дешевле)

3. **WiFi Triangulation**
   - Точность: ~2-5 м
   - Дальность: Зависит от роутера
   - **Рекомендация:** Для простых сценариев в помещении

**Выбор:** ESP32 с Bluetooth iBeacon - хорошее начало. Позже можно перейти на UWB для большей точности.

### Аудио система

**Микрофон:**
- USB микрофон (например, Logitech H390) - просто подключить
- MEMS микрофон + USB звуковая карта (например, Behringer UCA202)

**Динамик:**
- USB звуковая карта с выходом на динамик 4-8 Ом
- Или напрямую через Raspberry Pi аудио выход (3.5mm jack)

**Усилитель (опционально):**
- PAM8403 (стерео, 3W на канал)
- LM386 (моно, 1W)

## Примечания

1. **Лидар MOP3:** Возможно потребуется реверс-инжиниринг протокола. Проверьте доступные библиотеки или документацию.
2. **STM32 связь:** UART - самый простой вариант для начала.
3. **Питание:** Убедитесь в достаточной емкости батареи для работы минимум 1-2 часа.
4. **Охлаждение:** Raspberry Pi 5 может перегреваться под нагрузкой - используйте радиаторы или активное охлаждение.
