# Итоговый список выбранных компонентов

## ✅ Все выбранные компоненты (12 пунктов)

### 1. Контроллер

  - Cortex-M4, 100MHz
  - 512KB Flash, 128KB RAM

### 2. Коммуникация
- **USB-UART конвертер CP2102** (1 шт)
  - С проводами для подключения
  - Для подключения лидара к Beelink Mini PC

### 3. Соединительные элементы
- **JST коннекторы PH2.0 2P** (50 комплектов)
  - Белый штекер + основание
  - Длина провода: 150мм
  - Для подключения энкодеров и других компонентов

- **Провод 8-жильный 30AWG** (250м)
  - Луженый медный провод
  - Для сигнальных соединений

### 4. Источники питания
- **LiPo 12V 10000mAh** (1 шт)
  - Напряжение: 12V (3S, 12.6V при полном заряде)
  - Емкость: 10000mAh ✅
  - Размеры: 125×65×22мм
  - Разъем: 5.5×2.1мм (выход и зарядка)
  - **Время работы: 2-3 часа**

### 5. Преобразователи напряжения
- **LM2596S 8-40V→5V** (2 шт)
  - Понижающий преобразователь DC-DC
  - Для энкодеров и других 5V устройств

### 6. Энкодеры
- **E6B2-CWZ6C 1000P/R** (Omron) (2 шт)
  - Инкрементальный энкодер
  - Разрешение: 1000 P/R = 4000 counts/revolution
  - Напряжение: 5V (требует делители для STM32 3.3V)
  - Кабель: 2 метра

### 7. Моторы
- **25GA370 мотор-редуктор** (2 шт)
  - Постоянного тока
  - Напряжение: 12V
  - Без встроенных энкодеров

### 8. Драйверы моторов
- **DRV8833** (4 шт)
  - Для моторов постоянного тока
  - Ток: до 1.5A на канал
  - Нужно: 2 шт (запас: 2 шт)

### 9. Лидар
- **RPLiDAR ToF C1**
  - Комплект портативного лазерного сканера
  - Сканирование на 360°
  - Дальность: 12 метров

### 10. Камера
- **Logitech C920e 1080p**
  - USB подключение
  - Разрешение: 1080p
  - Встроенный микрофон

### 11. Шасси и корпус
- **Самодельное** (проектируется самостоятельно)
  - См. `CHASSIS_DESIGN_GUIDE.md`

### 12. Beelink Mini PC
- **Уже имеется** (не в списке покупок)
  - Intel N100/N95 или аналогичный
  - Минимум 8GB RAM

## ❌ Недостающие компоненты

### Критично (для работы):
1. **Преобразователь 12V→3.3V** (для STM32) - **~$3-5**
   - AMS1117-3.3 или LM2596S на 3.3V
   - Ток: минимум 500mA

2. **Преобразователь для Beelink Mini PC** - **~$10-15**
   - Если Beelink 12V: стабилизатор 12V→12V
   - Если Beelink 19V: step-up 12V→19V
   - Ток: минимум 2-3A

3. **Зарядное устройство LiPo 12V** (3S) - **~$10-20**
   - Напряжение: 12.6V (для 3S)
   - С балансировкой
   - Ток: 2-3A (для 10000mAh)

4. **Делители напряжения 5V→3.3V** (2 шт) - **~$2-5**
   - Для сигналов E6B2-CWZ6C (5V) → STM32 (3.3V)
   - Резисторы (10kΩ + 20kΩ) или готовые конвертеры

### Обязательно (для подключения):
5. **Кабели для моторов и питания** - **~$5-10**
   - Кабели для моторов (сечение 0.5-1mm²): ~40см на мотор
   - Кабели для питания (сечение 1-2mm²): ~50см
   - Разъем 5.5×2.1 для подключения аккумулятора (если нужно)

## 📊 Расчет времени работы

### Потребление (при 12V):
- Beelink Mini PC: ~10-30W (0.8-2.5A)
- STM32: ~0.1A
- Моторы (2x): ~1.5A каждый = 3A (пик)
- Энкодеры: ~0.2A
- Преобразователи: ~0.1A
- **Итого: ~4.2-6A при работе**

### Время работы с аккумулятором 10000mAh:
- При полной нагрузке (6A): **~1 час 40 минут**
- При средней нагрузке (4.5A): **~2 часа 13 минут**
- При минимальной нагрузке (4.2A): **~2 часа 23 минуты**

**✅ Отлично! Аккумулятор 10000mAh обеспечивает работу 2-3 часа.**

## 🔌 Схема подключения питания

```
Аккумулятор 12V 10000mAh (5.5×2.1)
    │
    ├─→ LM2596S #1 (12V→5V) → Энкодеры E6B2-CWZ6C (2 шт)
    │
    ├─→ LM2596S #2 (12V→5V) → Резерв/другие устройства
    │
    ├─→ Преобразователь 12V→3.3V → STM32 Nucleo
    │
    ├─→ Преобразователь 12V→12V/19V → Beelink Mini PC
    │   (зависит от модели Beelink)
    │
    └─→ DRV8833 (2 шт) → Моторы 25GA370 (2 шт)
```

## ⚠️ Важные замечания

1. ✅ **Аккумулятор 10000mAh** - отлично! Хватит на 2-3 часа работы

2. **Напряжение Beelink Mini PC** - критично проверить перед покупкой преобразователя
   - Если Beelink 12V: может работать напрямую от аккумулятора (через стабилизатор)
   - Если Beelink 19V: нужен step-up преобразователь 12V→19V

3. **Разъем аккумулятора 5.5×2.1** - стандартный разъем
   - Выход: для подключения к системе
   - Зарядка: для зарядного устройства

4. **Делители напряжения** - обязательно для энкодеров E6B2-CWZ6C
   - 5V сигналы энкодеров → 3.3V входы STM32

5. **DRV8833 (4 шт)** - запас на случай поломки (нужно только 2)

6. **JST коннекторы (50 шт)** - более чем достаточно для всех подключений
