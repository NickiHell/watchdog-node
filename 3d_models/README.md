# 3D Модели корпуса WatchDog Robot

Этот каталог содержит все файлы для создания и печати корпуса робота WatchDog.

## 📁 Структура

```
3d_models/
├── freecad_scripts/     # Python скрипты для FreeCAD
│   ├── parameters.py
│   ├── base_plate.py
│   ├── electronics_compartment.py
│   ├── lidar_tower.py
│   ├── camera_bracket.py
│   ├── side_panels.py
│   ├── top_cover.py
│   ├── cooling_duct.py
│   └── README.md
└── README.md           # Этот файл
```

## 🎯 Быстрый старт

1. Откройте FreeCAD
2. Перейдите в папку `freecad_scripts/`
3. Следуйте инструкциям в `freecad_scripts/README.md`

## 📐 Концепция корпуса

Корпус состоит из 7 основных деталей:

1. **Нижняя платформа** - основание для шасси и моторов
2. **Отсек электроники** - размещение Raspberry Pi, STM32, батареи
3. **Башня лидара** - поднятие лидара на высоту
4. **Кронштейн камеры** - установка камеры с регулировкой
5. **Боковые панели** - защита и вентиляция
6. **Верхняя крышка** - защита электроники
7. **Канал охлаждения** - активное охлаждение Raspberry Pi

## 🖨️ Печать деталей

После создания моделей в FreeCAD:

1. Экспортируйте каждую деталь в STL
2. Откройте в слайсере (Cura, PrusaSlicer, etc.)
3. Настройте параметры печати:
   - Материал: PLA+ или PETG
   - Заполнение: 20-30%
   - Толщина слоя: 0.2-0.3 мм
   - Поддержки: по необходимости

## 🔧 Сборка
