# 3D Модели корпуса WatchDog Robot

Этот каталог содержит FreeCAD скрипты для создания всех деталей корпуса робота WatchDog.

## 📁 Структура файлов

```
freecad_scripts/
├── parameters.py          # Параметры корпуса (размеры, зазоры)
├── create_all.py          # Мастер-скрипт для создания всех деталей
├── assemble.py            # Скрипт для сборки всех деталей в один документ
├── base_plate.py          # Нижняя платформа
├── electronics_compartment.py  # Отсек для электроники (Beelink Mini S)
├── lidar_tower.py         # Башня для лидара
├── camera_bracket.py      # Кронштейн для камеры
├── side_panels.py         # Боковые панели
├── top_cover.py           # Верхняя крышка
├── cooling_duct.py        # Канал охлаждения (для Beelink Mini S)
├── export_parameters.py  # Скрипт для экспорта параметров из модели обратно в код
└── README.md              # Этот файл
```

## 🚀 Использование

### Вариант 1: Создание всех деталей одной командой (РЕКОМЕНДУЕТСЯ)

Используйте мастер-скрипт `create_all.py` для создания всех деталей сразу:

1. Откройте FreeCAD
2. Откройте Python консоль (View → Panels → Python console)
3. Выполните команду:
```python
exec(open('/home/nickihell/Workspace/Projects/nickihell/watchdog-node/3d_models/freecad_scripts/create_all.py').read())
```

Скрипт создаст все 7 деталей корпуса в отдельных документах FreeCAD и выведет статистику.

### Вариант 2: Сборка всех деталей в один документ (для визуализации)

После создания всех деталей (`create_all.py`), вы можете собрать их в один документ для визуализации:

1. Откройте FreeCAD
2. Откройте Python консоль (View → Panels → Python console)
3. Выполните команду:
```python
exec(open('/home/nickihell/Workspace/Projects/nickihell/watchdog-node/3d_models/freecad_scripts/assemble.py').read())
```

Скрипт создаст документ `WatchDog_Assembly` со всеми деталями, собранными в правильном положении.

**Управление видом:**
- Вращение: средняя кнопка мыши
- Масштаб: колесо мыши
- Панорамирование: Shift + средняя кнопка мыши
- Подогнать вид: клавиша `0` или View → Standard views → Fit all

### Вариант 3: Запуск отдельных скриптов в FreeCAD

1. Откройте FreeCAD
2. Перейдите в меню **Macro → Macros...**
3. Нажмите **Create...** и создайте новый макрос
4. Скопируйте содержимое нужного скрипта в макрос
5. Запустите макрос (F6)

### Вариант 4: Запуск через Python консоль

1. Откройте FreeCAD
2. Откройте Python консоль (View → Panels → Python console)
3. Выполните команду:
```python
exec(open('/путь/к/скрипту.py').read())
```

### Вариант 5: Прямой запуск из командной строки

```bash
freecad -c /путь/к/скрипту.py
```

## ⚙️ Компоненты

Корпус разработан для следующих компонентов:

- **Beelink Mini S** (115×102×41 мм) - основной компьютер вместо Raspberry Pi
- **STM32F407 Nucleo** - контроллер моторов
- **Лидар Xiaomi MOP3** - навигация
- **Raspberry Pi Camera v3** - камера
- **Li-Po 12V 5000mAh** - аккумулятор
- **DRV8833** - драйверы моторов (2 шт)

## 📝 Настройка параметров

Перед созданием моделей отредактируйте файл `parameters.py`:

- Измените размеры под ваши компоненты
- Настройте зазоры и допуски
- Настройте размеры крепёжных отверстий

### Основные параметры:

```python
CASE_LENGTH = 320      # Длина корпуса (мм) - увеличено для Beelink Mini S
CASE_WIDTH = 280       # Ширина корпуса (мм) - увеличено для Beelink Mini S
CASE_HEIGHT = 220      # Высота корпуса (мм) - увеличено для Beelink Mini S
WALL_THICKNESS = 3     # Толщина стенок (мм)
MINIPC_LENGTH = 115    # Beelink Mini S длина (мм)
MINIPC_WIDTH = 102     # Beelink Mini S ширина (мм)
MINIPC_HEIGHT = 41     # Beelink Mini S высота (мм)
```

## 🖨️ Экспорт для 3D печати

После создания модели в FreeCAD:

1. Выберите объект в дереве проекта
2. Перейдите в меню **File → Export**
3. Выберите формат **STL** или **STEP**
4. Сохраните файл

### Рекомендации для печати:

- **Материал**: PLA+ или PETG
- **Толщина стенок**: 2-3 мм (уже учтено в моделях)
- **Заполнение**: 20-30%
- **Слой**: 0.2-0.3 мм
- **Поддержки**: минимальные (ориентируйте детали правильно)

## 📦 Детали корпуса

### 1. Base Plate (Нижняя платформа)
- Размер: 320×280×8 мм (увеличено для Beelink Mini S)
- Отверстия для крепления моторов (стандарт Pololu 37D)
- Отверстия для крепления к шасси

### 2. Electronics Compartment (Отсек электроники)
- Отсеки для Beelink Mini S, STM32, батареи, драйверов
- Вентиляционные отверстия
- Кабельные каналы
- Размеры увеличены для размещения Beelink Mini S (115×102×41 мм)

### 3. Lidar Tower (Башня лидара)
- Высота: 100 мм
- Диаметр основания: 120 мм
- Крепление для лидара MOP3

### 4. Camera Bracket (Кронштейн камеры)
- Регулируемый угол наклона
- Крепление для Raspberry Pi Camera v3
- Защита кабеля

### 5. Side Panels (Боковые панели)
- Левая и правая панели
- Вентиляционные отверстия
- Крепления для динамика/микрофона

### 6. Top Cover (Верхняя крышка)
- Защита электроники
- Отверстие для башни лидара
- Кабельные отверстия
- Платформа для USB камеры в углу

## 🔄 Экспорт параметров из модели обратно в скрипты

Если вы вносите изменения в модели вручную в FreeCAD, используйте скрипт `export_parameters.py` для экспорта параметров:

1. Откройте FreeCAD
2. Откройте Python консоль (View → Panels → Python console)
3. Выполните:
```python
exec(open('/home/nickihell/Workspace/Projects/nickihell/watchdog-node/3d_models/freecad_scripts/export_parameters.py').read())
```

4. Экспортируйте параметры:
```python
# Экспорт одного документа
export_document('WatchDog_BasePlate')

# Экспорт всех открытых документов
export_all_documents()

# Экспорт позиций из сборки (для assemble.py)
export_assembly_positions('WatchDog_Assembly1')

# Экспорт в файл
export_parameters_to_file('WatchDog_BasePlate')
```

Скрипт выведет все параметры объектов (размеры, позиции, повороты, цвета) в удобном формате, который можно скопировать в исходные скрипты.

### 7. Cooling Duct (Канал охлаждения)
- Для активного охлаждения Beelink Mini S
- Крепление для вентилятора 40×40 мм
- Направляющие лопатки
- Размеры адаптированы под Beelink Mini S

## 🔧 Сборка корпуса

### Порядок сборки:

1. **Нижняя платформа**
   - Установите на шасси
   - Закрепите моторы

2. **Отсек электроники**
   - Установите на нижнюю платформу
   - Разместите компоненты в отсеках
   - Подключите провода

3. **Канал охлаждения**
   - Установите над Raspberry Pi
   - Закрепите вентилятор

4. **Верхняя крышка**
   - Установите на отсек электроники
   - Закрепите винтами

5. **Башня лидара**
   - Установите на верхнюю крышку
   - Закрепите лидар

6. **Кронштейн камеры**
   - Установите на боковую панель
   - Закрепите камеру

7. **Боковые панели**
   - Установите слева и справа
   - Закрепите винтами

## 🐛 Решение проблем

### Скрипт не запускается

- Убедитесь, что все файлы находятся в одной папке
- Проверьте, что `parameters.py` доступен
- Проверьте версию FreeCAD (рекомендуется 0.20+)

### Модель не создаётся

- Проверьте параметры в `parameters.py`
- Убедитесь, что размеры логичны (не отрицательные, не слишком большие)
- Проверьте консоль FreeCAD на ошибки

### Отверстия не создаются

- Увеличьте `SCREW_HOLE_DIAMETER` в параметрах
- Проверьте, что отверстия не выходят за границы детали
