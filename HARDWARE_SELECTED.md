# Выбранные компоненты для WatchDog Robot

## ✅ Выбранное оборудование

### Основной компьютер
- **Beelink Mini PC** (Intel N100/N95 или аналогичный)
  - Минимум 8GB RAM
  - SSD 128GB+
  - Напряжение питания: требуется проверить (12V или 19V)

### Контроллер движения
- **NUCLEO-F411RE** (STM32F411RE)
  - Cortex-M4, 100MHz
  - 512KB Flash, 128KB RAM
  - Поддерживается проектом

### Драйверы моторов
- **DRV8833** (4 шт) ✅ (запас на случай поломки)
  - Для моторов постоянного тока
  - Ток: до 1.5A на канал
  - Нужно: 2 шт (по одному на мотор)
  - Запас: 2 шт

### Моторы
- **25GA370** (2 шт)
  - Мотор-редуктор постоянного тока
  - Напряжение: 12V
  - Без встроенных энкодеров

### Энкодеры
- **E6B2-CWZ6C 1000P/R** (Omron) (2 шт)
  - Инкрементальный энкодер
  - Разрешение: 1000 P/R = 4000 counts/revolution (quadrature)
  - Напряжение: 5V (требует делители для STM32 3.3V)
  - Кабель: 2 метра

### Лидар
- **RPLiDAR ToF C1**
  - Комплект портативного лазерного сканера
  - Требует проверки совместимости протокола
  - См. `docs/RPLIDAR_TOF_C1_SETUP.md`

### Камера
- **Logitech C920e 1080p**
  - USB подключение
  - Разрешение: 1080p
  - Встроенный микрофон
  - Полностью совместимо с проектом

### Источники питания
- **LiPo 12V 10000mAh**
  - Напряжение: 12V (3S, 12.6V при полном заряде)
  - Емкость: 10000mAh ✅ (отлично! хватит на 3-4 часа работы)
  - Размеры: 125×65×22мм
  - Разъем: 5.5×2.1мм (выход и зарядка)
  - Жесткий корпус

### Преобразователи напряжения
- **LM2596S 8-40V → 5V** (2 шт)
  - Понижающий преобразователь DC-DC
  - Для энкодеров и других 5V устройств

### Шасси и корпус
- **Самодельное** (проектируется самостоятельно)
  - Рекомендуемые размеры: 300×280×180мм (без лидара)
  - Место для всех компонентов

### Соединительные элементы
- **USB-UART конвертер CP2102** ✅
  - С проводами для подключения
  - Для лидара и возможно STM32
  
- **JST коннекторы PH2.0 2P** (50 комплектов) ✅
  - Для подключения энкодеров, моторов и других компонентов
  - Белый штекер + основание
  - Длина провода: 150мм
  
- **Провод 8-жильный 30AWG** (250м) ✅
  - Луженый медный провод
  - Для сигнальных соединений
  - Для монтажа и соединений

## ❌ Недостающие компоненты

### Критично (для работы):

1. **Преобразователь 11.1V → 3.3V** (для STM32) - **~$3-5**
   - AMS1117-3.3 или LM2596S на 3.3V
   - Ток: минимум 500mA

2. **Преобразователь для Beelink Mini PC** - **~$10-15**
   - Если Beelink 12V: стабилизатор 11.1V→12V (step-up)
   - Если Beelink 19V: step-up 11.1V→19V
   - Ток: минимум 2-3A

3. **Зарядное устройство LiPo 3S** - **~$10-20**
   - Напряжение: 12.6V (для 3S)
   - С балансировкой
   - Ток: 1-2A (для 2200mAh)

4. **Делители напряжения 5V→3.3V** (2 шт) - **~$2-5**
   - Для сигналов E6B2-CWZ6C (5V) → STM32 (3.3V)
   - Резисторы (10kΩ + 20kΩ) или готовые конвертеры уровня

### Обязательно (для подключения):

5. **Перемычки и кабели** - **~$5-10** (частично покрыто)
   - Кабели для моторов (сечение 0.5-1mm²): ~40см
   - Кабели для питания (сечение 1-2mm²): ~50см
   - Разъем 5.5×2.1 для подключения аккумулятора (если нужно)

### Рекомендуется:

7. ✅ **Аккумулятор 10000mAh** - **ОТЛИЧНО!**
   - Достаточно для работы 3-4 часа (при нагрузке 4.5-6A)
   - Не требуется дополнительный аккумулятор

### Опционально (для полного функционала):

8. **USB микрофон** - **~$15** (если в веб-камере нет хорошего микрофона)
9. **USB звуковая карта** - **~$10**
10. **Динамик 4-8 Ом** - **~$5**
11. **IMU датчик MPU6050** - **~$5** (для улучшения навигации)

## 📊 Расчет потребления и времени работы

### Потребление (при 12V):
- Beelink Mini PC: ~10-30W (0.8-2.5A при 12V)
- STM32: ~0.1A
- Моторы (2x): ~1.5A каждый = 3A (пик)
- Энкодеры: ~0.2A
- Преобразователи: ~0.1A
- **Итого: ~4.2-6A при работе**

### Время работы с аккумулятором 10000mAh:
- 10000mAh / 6A ≈ **1 час 40 минут** (при полной нагрузке)
- 10000mAh / 4.5A ≈ **2 часа 13 минут** (при средней нагрузке)
- 10000mAh / 4.2A ≈ **2 часа 23 минуты** (при минимальной нагрузке)

**✅ Отлично!** Аккумулятор 10000mAh обеспечивает работу 2-3 часа - более чем достаточно!

## 🔌 Схема подключения питания

```
Аккумулятор 12V 10000mAh (5.5×2.1)
    │
    ├─→ LM2596S #1 (12V→5V) → Энкодеры E6B2-CWZ6C (2 шт)
    │
    ├─→ LM2596S #2 (12V→5V) → Резерв/другие устройства
    │
    ├─→ Преобразователь 12V→3.3V → STM32 Nucleo
    │
    ├─→ Преобразователь 12V→12V/19V → Beelink Mini PC
    │   (зависит от модели Beelink)
    │
    └─→ DRV8833 (2 шт) → Моторы 25GA370 (2 шт)
```

## ⚠️ Важные замечания

1. ✅ **Аккумулятор 10000mAh** - отлично! Хватит на 2-3 часа работы

2. **Напряжение Beelink Mini PC** - критично проверить перед покупкой преобразователя (12V или 19V)
   - Если Beelink 12V: может работать напрямую от аккумулятора (через стабилизатор)
   - Если Beelink 19V: нужен step-up преобразователь 12V→19V

3. **Разъем аккумулятора 5.5×2.1** - стандартный разъем, удобно для подключения
   - Выход: 5.5×2.1мм (для подключения к системе)
   - Зарядка: 5.5×2.1мм (для зарядного устройства)

3. **Делители напряжения** - обязательно для энкодеров E6B2-CWZ6C (5V сигналы) → STM32 (3.3V входы)

4. **Шасси** - спроектировать с учетом:
   - Размеры Beelink Mini PC (~115×102×41мм)
   - Размеры STM32 Nucleo (~70×52×18мм)
   - Размеры аккумулятора (~125×65×22мм) - плоский, удобно размещать
   - Крепление лидара сверху (высота ~70мм)
   - Вентиляция для Beelink Mini PC

5. **LM2596S** - можно один настроить на 3.3V для STM32, или купить отдельный модуль

6. **JST коннекторы PH2.0** - удобно для подключения энкодеров и других компонентов
   - 50 комплектов - более чем достаточно
   - Длина провода 150мм - удобно для подключений

7. **Провод 8-жильный 30AWG** - для сигнальных соединений
   - 250м - более чем достаточно
   - Луженый - удобно для пайки
