## 🚀 Быстрый старт

### Запуск тестов
```bash
pytest ros2_ws/src/ -v
```

### Сборка пакетов
```bash
cd ros2_ws
colcon build --symlink-install
source install/setup.bash
```

### Запуск с Docker
```bash
docker-compose up
```

### Использование новых утилит
```python
from watchdog_common import (
    get_logger,
    DiagnosticPublisher,
    ConfigValidator,
    retry,
    SecurityValidator
)
```
